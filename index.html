About Me

I am Haoran Yu, a researcher and data science professional pursuing a Bachelor of Science in Mathematics at the University of Toronto, with minors in Statistics and Economics. My academic background in probability, statistics, financial economics, data analysis, and machine learning has provided me with a strong foundation in quantitative methods. I leverage these skills to develop robust analytical frameworks and predictive models for complex real-world challenges, while continuously expanding my expertise through ongoing research and projects.

I am proficient in Python (including libraries such as NumPy, pandas, Matplotlib, PyTorch, SciPy, and scikit-learn), R, SAS, and SQL.

University of Toronto Undergraduate Course Notes

This project curates comprehensive course notes from my classes at the University of Toronto, primarily covering mathematics, economics, and statistics. Prepared in LaTeX, these notes are updated regularly throughout my undergraduate journey, with earlier versions revised as needed. Please note that course content may differ from session to session, so the notes are tailored to the curriculum I experienced—meaning some proofs, topics, or details might be incomplete. As I continue to learn and grow, some definitions, theorems, and lemmas may not be entirely accurate. This project is currently ongoing, and you can expect more notes to be added in the Notes section of this website.

Regression Analysis on Insurance Premium

This project performs a comprehensive regression analysis on insurance premium data to identify key factors influencing premium pricing. The analysis employs statistical methods to evaluate the relationships between various predictors and premium amounts, offering insights that could help refine pricing strategies. The work includes data cleaning, exploratory data analysis, and building regression models using R. This project demonstrates the practical application of data science in the insurance industry.

View Project on GitHub

Detection of YouTube Spam Comments

This course project for STA314H1 focuses on detecting spam comments on YouTube. It involves cleaning and processing text data, extracting relevant features, and applying machine learning algorithms to classify comments as spam or non-spam. The project demonstrates the use of statistical methods and natural language processing techniques to address real-world challenges in content moderation.

View Project on GitHub

Practice: Research in Biodiversity of National Park

This practice project investigates the biodiversity within a national park using a combination of statistical analysis. It aims to identify key species, map their distribution, and assess the influence of environmental factors on ecosystem diversity. The findings contribute valuable insights for conservation efforts and ecological studies.
